About

During lockdown 2020, I watched a lot of Survivor.

I've created a database of player information pulled from the first 40 seasons of Survivor and crunched some numbers. Here are some of my findings.

The code to the project, as well as a more technical breakdown to how I did everything, plus the database can be found here. If anyone wants to keep this project up to date as new seasons air, I've included information in the Github on how to do that.

Acknowledgements

I started this database with data from Survivor Wiki, and added addition information to it.
